# CP1 



### 파일 접근법
1. 원하는 사용자의 폴더 (e.g. haesol)
2. 원하는 주의 폴더 (e.g. week1)
3. 해당 주차의 문제 번호 (e.g. week1_asgmt_1)

### 이용 가이드
1. 문제를 최대한 풀어본 뒤
2. 도저히 방법이 떠오르지 않으면
3. 다른 사람의 풀이를 확인해 보고(복사X)
4. 스스로 코드를 다시 작성해본다

※변수명은 다시 작성해보기!!

### 명명 규칙
- 개인 폴더: 영문 이름 (e.g. haesol)
- 각 주별 폴더 생성 (e.g. week1)
- 해당 주의 특정 번호 (e.g. week1_asgmt_1)
